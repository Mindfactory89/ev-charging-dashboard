## Vorschlag fuer die erste Release-Beschreibung

```md
## Highlights

- Private EV charging dashboard with local PostgreSQL
- Beginner-friendly Docker Compose setup
- Yearly analytics, monthly reports, forecasts, and year comparison
- Smart insights, SoC-based charging analysis, and inline session editing
- Demo mode for testing without a database

## Included

- Public-ready `.env.example`
- Generic VPS deploy scripts
- Beginner Docker setup via `docker-compose.beginner.yml`
- MIT license

## Notes

- Users run the app with their own database and their own data
- No personal server access or private credentials are included
```
